# Movie Poster Guessing Game Setup

## How to Add Images

1.  **Locate the Image Folder**:
    Go to `games/movie-poster/img/` in this repository.

2.  **Add Your Images**:
    Place your movie poster images (JPG or PNG) into this folder.

3.  **Update the Code**:
    Open `games/movie-poster/script.js`.
    Find the `posters` array at the top of the file:

    ```javascript
    const posters = [
        'img/poster1.jpg',
        'img/poster2.jpg',
        'img/poster3.jpg',
        // Add your filenames here
    ];
    ```
    
    Update the list to match the filenames of the images you added.

4.  **Play**:
    Open `games/movie-poster/index.html` in your browser!
